## E-Commerce Revenue Analysis

## Project Overview:-

This project analyzes sales, profit, customer behavior, product performance, and regional trends using the Superstore dataset. The objective is to identify key business opportunities, improve profitability, and support data-driven decision-making through interactive Tableau dashboards.

## Dataset Files:-

\- orders\_clean.csv

\- returns\_clean.csv

\- people\_clean.csv

## Tools Used:-


\- PostgreSQL

\- Tableau Public

\- CSV Datasets

\- SQL

\- Data Visualization


## Project Workflow:-

Data Analysis (Python)

\* Loaded and explored the dataset

\* Checked data quality and missing values

\* Performed exploratory data analysis (EDA)

\* Analyzed sales, profit, customers, products, and regions

\* Generated business insights

\* Prepared data for visualization

Dashboard 2 — Profitability \& Operational Insights


\- Sub-Category Profitability Analysis

\- Discount Impact Analysis

\- Ship Mode Usage \& Sales Analysis

\- Business Key Findings


## Key Business Questions Solved:-


\* Which category generated the highest sales?

\* Which category generated the highest profit?

\* Which sub-category caused losses?

\* How do discounts affect profitability?

\* Which region performed best?

\* Which ship mode generated the most orders and sales?

\* Which customers generated the highest sales?

\* How did yearly sales and profit perform?


## Key Insights:-


\* Technology generated the highest sales and profit.

\* Some sub-categories contributed to significant losses despite strong sales.

\* Higher discounts often reduced profitability.

\* Certain regions consistently outperformed others in both sales and profit.

\* A small group of customers contributed a large portion of total sales.

\* Sales and profit trends showed seasonal patterns and growth opportunities.
